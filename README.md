# @andreaemme/ReactElements

ReactElements is a simple React component library that you can use to build your application.

## Install package
```sh
npm i @andreaemme/reactelements
```

## Available components
ReactElements includes:
- Spinner
- Avatar
- Tag Cloud
- Masonry Gallery

Package Structure:

```sh
+-- _Components
|   +-- Avatar
|   +-- MasonryGallery
|   +-- Spinner
|   +-- TagCloud
```
